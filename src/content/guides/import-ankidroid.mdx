---
title: "Cách import deck .apkg trên AnkiDroid"
slug: "import-ankidroid"
tags: ["ankidroid","import","android"]
guideType: "Cơ bản"
---

1. Tải file `.apkg` về điện thoại của bạn.
2. Mở ứng dụng AnkiDroid, vào Decks > dấu “+” > Import.
3. Chọn file `.apkg` đã tải. Đợi quá trình import hoàn tất.
4. Đồng bộ với AnkiWeb nếu cần.
