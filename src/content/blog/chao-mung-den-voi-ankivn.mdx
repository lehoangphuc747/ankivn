# Chào mừng đến với AnkiVN! 🎉

Xin chào và chào mừng bạn đến với **AnkiVN** - nền tảng chia sẻ bộ thẻ Anki chất lượng cao dành cho cộng đồng học tập Việt Nam!

## Giới thiệu về AnkiVN

AnkiVN được ra đời với *sứ mệnh* kết nối và hỗ trợ cộng đồng người Việt trong hành trình học tập. Chúng tôi cung cấp một thư viện phong phú các bộ thẻ Anki được ***kiểm duyệt kỹ lưỡng*** để đảm bảo chất lượng tốt nhất.

### Tại sao nên sử dụng Anki?

Anki là một công cụ học tập mạnh mẽ dựa trên nguyên lý **Spaced Repetition** (lặp lại ngắt quãng). Đây là phương pháp đã được khoa học chứng minh hiệu quả trong việc:

1. **Ghi nhớ lâu dài**: Tăng khả năng lưu trữ thông tin trong bộ nhớ dài hạn
2. **Tối ưu thời gian**: Chỉ cần 15-30 phút mỗi ngày
3. **Học tập hiệu quả**: Tập trung vào những gì bạn chưa thuộc
4. **Linh hoạt**: Có thể học mọi lúc, mọi nơi

### Danh mục nội dung phong phú

AnkiVN cung cấp các bộ thẻ đa dạng trong nhiều lĩnh vực:

#### 🌍 Ngoại ngữ
- **Tiếng Anh**: Từ vựng, ngữ pháp, IELTS, TOEIC
- **Tiếng Nhật**: JLPT N5-N1, Kanji, từ vựng
- **Tiếng Trung**: HSK, từ vựng cơ bản đến nâng cao
- **Các ngôn ngữ khác**: Hàn, Pháp, Đức, Nga...

#### 📚 Giáo dục
- **THCS**: Toán, Lý, Hóa, Văn
- **THPT**: Các môn thi THPT Quốc gia
- **Đại học**: Chuyên ngành cụ thể

#### 💼 Chuyên ngành
- **Y Dược**: Thuật ngữ y khoa, dược học
- **IT**: Lập trình, thuật ngữ công nghệ
- **Kinh tế**: Tài chính, kế toán, marketing

## Hướng dẫn sử dụng cơ bản

### Bước 1: Tải và cài đặt Anki

```bash
# Download Anki từ trang chính thức
https://apps.ankiweb.net/

# Hoặc sử dụng package manager (macOS)
brew install --cask anki
```

### Bước 2: Import bộ thẻ từ AnkiVN

1. Truy cập [AnkiVN.com](https://ankivn.com)
2. Chọn bộ thẻ phù hợp
3. Click vào nút **"Tải xuống"**
4. Mở Anki và chọn `File > Import`
5. Chọn file `.apkg` vừa tải về

### Bước 3: Bắt đầu học

> **💡 Mẹo**: Hãy học đều đặn mỗi ngày, tốt hơn là học ít nhưng thường xuyên thay vì học nhiều nhưng không đều!

## Code Example - Tạo thẻ Anki tùy chỉnh

Bạn có thể tạo thẻ Anki với HTML và CSS:

```html
<!-- Front Template -->
<div class="card front">
  <div class="word">{{Word}}</div>
  <div class="phonetic">{{Phonetic}}</div>
</div>

<!-- Back Template -->
<div class="card back">
  <div class="word">{{Word}}</div>
  <div class="meaning">{{Meaning}}</div>
  <div class="example">{{Example}}</div>
</div>
```

```css
/* Styling */
.card {
  font-family: Arial, sans-serif;
  text-align: center;
  padding: 20px;
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  color: white;
  border-radius: 10px;
}

.word {
  font-size: 2em;
  font-weight: bold;
  margin-bottom: 10px;
}

.meaning {
  font-size: 1.2em;
  margin: 15px 0;
}
```
